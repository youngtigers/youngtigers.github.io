# Thomas Tao Yang Website

Static academic website for GitHub Pages.

## Files

- `index.html` - tabbed one-page website.
- `styles.css` - responsive styling.
- `profile.jpg` - profile image.
- `CV-TaoYang_use.pdf` - CV download and preview.

## Publish With GitHub Pages

Upload these files to a GitHub repository, then enable Pages from the repository settings using the main branch and root folder.
